English

RC4 Speed 2.5s is freeware

WHAT IS NEW?

+ RC4 Speed now use cryptographic Hash Routines (MD5 and SHA)
+ 'Options' have two new points to select the Hash Routine
+ New Splash Screen

What does this program?

It encrypts your personal files in a way that nobody can access your files without knowing the right password. 


Where does this cryptic name come from?

RC4 Speed uses the RC4 Stream Cipher to encrypt/decrypt your files. 


How does I encrypt/decrypt my files?

There are two possibilities to en/decrypt files with RC4 Speed:

1. You enter your Password and click on Encrypt (Decrypt) and select your file to encrypt/decrypt.

2. You enter your Password and Drag & Drop your file(s) to the Encrypt (Decrypt) Button



Technical Information
---------------------

I use Visual Basic 5.0 to make RC4 Speed. It encrypts/decrypts files with max. 650 KB/s
(P200 Mhz). The Source Code is an (encrypted) part of this zip. If you want to know the password just read the About Section.

RC 4 Speed 2.5s ist Freeware


WAS IST NEU?

+ RC4 Speed benutzt jetzt kryptographisch sichere Hashroutinen
+ Im 'Option' Menü gibt es einen Punkt um die Hash Routinen auszuwählen
+ New Startup Logo


Was tut dieses Programm?

Es verschlüsselt ihre persönlichen Dateien so sicher, dass niemand sie ohne das richtige Passwort entschlüsseln kann! 


Woher kommt diese seltsame Name?

RC4 Speed benutzt den sogenannten RC4 Algorithmus zum ver/entschlüsseln.


Wie ver/entschlüssel ich meine Dateien?

Sie haben 2 Möglichkeiten um mit RC4 Speed eine Datei zu ver/entschlüsseln:

1. Sie geben Ihr Passwort ein und drücken auf Encrypt bzw. Decrypt und wählen ihre Datei,
die sie ver/entschlüsseln wollen

2. Sie geben ihr Passwort ein und ziehen per Drag & Drop eine (oder mehre) Dateien auf Encrypt bzw.
Decrypt
